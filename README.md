# Задание 2: Работа со строками
# Вариант 6
## "Создайте рекурсивную функцию для подсчета количества гласных букв в строке"
## Как работает алгоритм
В функции ```count_vowels``` строка проверяется рекурсивно, начиная с первого символа. Если строка пуста (базовый случай), возвращается ```0```. Иначе извлекается первый символ и оставшаяся часть строки. Если первый символ в нижнем регистре содержится в строке ```'aeiou'```, к результату рекурсивного вызова от остатка строки прибавляется ```1```, иначе возвращается только результат рекурсивного вызова. Процесс повторяется, пока строка не станет пустой. 
 
В функции ```count_vowels``` на C++ используется дополнительный параметр ```index``` со значением по умолчанию ```0```. Если ```index``` больше или равен длине строки (базовый случай), возвращается ```0```. Иначе текущий символ ```s[index]``` проверяется на совпадение с любой из гласных букв вручную, учитывая и строчные, и заглавные варианты. Если символ является гласной, к рекурсивному вызову с ```index + 1``` прибавляется ```1```, иначе — ```0```. Рекурсия продолжается до выхода за пределы строки. 

В Java-методе ```countVowels``` используется перегрузка: основной вызов передаёт строку и начинает с индекса ```0```. Если ```index``` достиг или превысил длину строки, возвращается ```0``` (базовый случай). Иначе символ по текущему индексу извлекается через ```charAt```, и его наличие проверяется в строке ```"aeiouAEIOU"``` с помощью ```indexOf```. Если ```indexOf``` возвращает значение не ```-1```, символ считается гласной, и к рекурсивному вызову с ```index + 1``` добавляется ```1```, иначе — ```0```. Рекурсия последовательно обходит строку слева направо, пока не завершится по достижении конца строки.
## На выходе в каждом коде получается: "3"
## Временная сложность
O(n)
## Почему именно такая сложность?
Каждый символ обрабатывается ровно один раз (один рекурсивный вызов на символ). Глубина рекурсии — n, но работа на каждом уровне — O(1).
